# Library records


What sort of data might we want to model for a library record?

* Author
  - Last name (string)
  - (maybe) middle initial (char or string)
  - First name (string)

* Book
  - title (string)
  - author (Author)
  - genre (?)
  - number of pages (int or unsigned int)
  - publishing date (Date or int)
  - location coding (string ?)
